# About SharM
SharMは楽曲共有サービスです。
このプロジェクトでは、以下のようなアプリケーションの機能を実装しています。

＊投稿機能

＊投稿にいいねをつける機能

＊認証機能

＊画像アップロード機能

開発環境はAWSを使用。
